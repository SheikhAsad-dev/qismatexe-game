# Qismatexe

A philosophical choice-based game in C exploring memory manipulation, destiny, and existential themes.

> "We are the stories we choose to remember, and the ones we dare to forget."

## Story Overview
You play as **Dr. Klein Morretti**, a neurosurgeon who can edit and erase traumatic memories. Your clinic, "The Vault," offers hope to the broken-hearted, but at what cost? As you confront patients, detectives, and your own past, you'll discover that every memory you alter changes not just their lives, but your own philosophical destiny.

## Game Features
- 6 immersive story scenes with moral dilemmas
- Multiple Philosophical Endings based on your choices:
- Nihilism Path: Embrace the meaninglessness of existence
- Absurdism Path: Find meaning in the struggle itself  
- Power Path: Become the architect of human experience
- Real-time text animation and colored console output
- Branching narrative with consequence tracking system

## Technologies Used
- C Programming Language
- Windows.h library for console manipulation
- Git for version control
